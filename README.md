## How to use

Running the server:

```sh
dotnet run --urls=http://localhost:8001 --project SignalrServerWorker/SignalrServerWorker.csproj
```

Running the client:

```sh
dotnet run --project SignalrClient/SignalrClient.csproj 8001
```
